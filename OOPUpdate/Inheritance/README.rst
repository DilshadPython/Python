Exception hierarchy

The class hierarchy for built-in exceptions is:

BaseException
  +── KeyboardInterrupt
  +── Exception
      +── ArithmeticError
      │    +── ZeroDivisionError
      +── AssertionError
      +── AttributeError
      +── BufferError
      +── EOFError
      +── ImportError
      │    +── ModuleNotFoundError
      +── LookupError
      │    +── KeyError
      +── MemoryError
      +── SyntaxError
      │    +── IndentationError
      +── ValueError
