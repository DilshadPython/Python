Numpy is a Linear Algebra Library for Python, the reason it is so important for Data Science with python is that almost all of the libraries in the PyData Ecosystem rely on NumPy as one
of their main building blocks.
Numpy is also incredibly fast, as it has bindings to C libraries.


- Numpy arrays are the main way we will use Numpy throughout the course.
- Numpy arrays essentially come in two flavors: 
    - Vectors
    - Matrices
- Vectors are strictly 1 - d arrays and matrices are 2 - d arrays (But you should note a matrix can still have only one row or one column)
